Генератор превью для шаринга артистов

Создаёт для каждого артиста статический файл artist/<slug>.html — копию текущего artists.html, но с подставленными og:title / og:image / og:description под конкретного артиста. Скачивает всё одним zip-архивом.

После скачивания: распакуй архив в корень проекта (появится/пополнится папка artist/) и сделай firebase deploy --only hosting. Перезапускай генератор каждый раз, когда добавляешь артиста или меняешь его имя/фото/описание.
